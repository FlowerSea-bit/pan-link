# 纯前端收集用户反馈的方法

## 🎯 已实现的方案

### 1. 本地存储 + 管理后台
- ✅ **优点**：简单易用，无需配置
- ❌ **缺点**：只能在同一台电脑上查看
- 📍 **使用方法**：打开 `admin.html` 查看反馈

### 2. 邮件发送（已集成）
- ✅ **优点**：直接发送到你的邮箱
- ❌ **缺点**：依赖用户的邮件客户端
- 📍 **配置方法**：修改代码中的 `your-email@example.com`

## 🛠️ 具体配置步骤

### 企业微信机器人配置（最推荐）

1. **创建企业微信群**
   - 下载企业微信APP
   - 创建一个群聊

2. **添加机器人**
   - 群设置 → 群机器人 → 添加机器人
   - 选择"自定义机器人"
   - 复制 Webhook 地址

3. **修改代码**
   ```javascript
   // 在 app.js 和 search.js 中找到这行：
   const webhookUrl = 'YOUR_WEBHOOK_URL_HERE';
   
   // 替换为你的实际地址：
   const webhookUrl = 'https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的key';
   ```

4. **测试**
   - 用户提交反馈后，群里会收到消息通知

### Server酱微信推送配置

1. **注册 Server酱**
   - 访问 https://sct.ftqq.com/
   - 微信扫码登录

2. **获取 SendKey**
   - 复制你的 SendKey

3. **修改代码**
   ```javascript
   // 添加到 sendToWebhook 函数中
   const serverChanUrl = `https://sctapi.ftqq.com/你的SendKey.send`;
   
   fetch(serverChanUrl, {
     method: 'POST',
     headers: { 'Content-Type': 'application/json' },
     body: JSON.stringify({
       title: '新的资源反馈',
       desp: `类型：${data.type}\n资源：${data.resourceName}\n描述：${data.content}`
     })
   });
   ```

## 💡 使用建议

1. **多方案结合**：同时使用本地存储 + 即时通知
2. **备份重要**：定期导出反馈数据
3. **隐私保护**：不要在代码中暴露敏感信息
4. **用户体验**：确保提交成功有明确反馈
